Area Formulas
  • Rectangle: A = length × width
  • Square: A = side²
  • Triangle: A = ½ × base × height
  • Circle: A = π × radius²
  • Parallelogram: A = base × height
  • Trapezoid: A = ½ × (a + b) × height
  • Ellipse: A = π × a × b
  • π ≈ 3.14159
